# weather-tracker
This project tracks the weather near Yosemite's Wawona campsite. This project was made so people can use it to plan camping trips to Yosemite when the weather is good. I made this project as part of my Intro to Data Science class. The script basically uses the Open Meteo API and fetches the min and max temperature of where your destination is from the previous 5 years of your input dates. It also fetches a 7 day forecast for temperature on the same place. You can use the two data frames to make an educated decision on when to go. However, the script will return an error if your dates include February 29th. You can try to work around it by using information from February 28th and March 1st. You can adjust the longitude and latitude and the start and end date inside the code to analyze and make decisions on where you want to go. Just search for the coordinates of your destination, put them in the script, and also put the month and start and end day into the script. The script will then fetch past weather data and forecast the next 7 days so you can make your desicion on when to visit.
